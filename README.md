## рҹҡҖ Getting Started

### Prerequisites
No build tools or backend required вҖ” this is a **pure front-end** static website.

### Run Locally

1. **Clone or Download** the project:
   ```bash
   git clone https://github.com/divyasolanki07/TastyBites-Food
   ```

2. **Open in Browser:**
   Simply double-click `food.html` or open it with your browser:
   ```
   food/food.html
   ```

3. **Or use Live Server** (VS Code):
   - Install the [Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
   - Right-click `food.html` вҶ’ **Open with Live Server**
